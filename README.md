# Dingus
Dingus mod for the popular vr game Gorilla Tag!

Adds: 

- Kittuh :3
  - Picking Up
  - Throwing
  - Networking for all users with the mod
  - Pick up and press X/B to mute his music :3

Requirements: 

 - Utilla Mod (off MMM)
 - This mod
 - BepinEx
 - Gorilla Tag
